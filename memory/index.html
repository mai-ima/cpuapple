UltraFabric との
深い統合

AX Memory は AXIOM 独自の UltraFabric ダイ間インターコネクト技術と緊密に統合されています。X1 Ultra と組み合わせることで、システム全体のレイテンシを最小化しながら最大帯域幅を引き出します。

CoWoS-L パッケージング技術により、メモリダイと SoC を同一パッケージ内に集積。信号伝播遅延をピコ秒単位で制御します。
