プライバシーポリシー

最終更新日：2026年5月30日 / 施行日：2026年1月1日

AXIOM Silicon 株式会社（以下「当社」といいます）は、お客様の個人情報の保護を重要な責務と認識し、個人情報の保護に関する法律（個人情報保護法）およびその他関連法令を遵守し、以下のとおりプライバシーポリシーを定めます。

第1条　定義

本ポリシーにおいて使用する用語の意義は、個人情報保護法に定めるところによるほか、以下のとおりとします。

  • 「個人情報」とは、生存する個人に関する情報であって、氏名・生年月日・メールアドレス・その他の記述等により特定の個人を識別することができる情報をいいます。
  • 「本サービス」とは、当社が提供するウェブサイト・製品・SDK・API・その他デジタルサービスの総称をいいます。
  • 「利用者」とは、本サービスを利用するすべての方をいいます。

第2条　収集する情報の種類

当社は、本サービスの提供に際して、以下の情報を収集することがあります。

（1）利用者が直接提供する情報

  • 氏名・会社名・部署名・役職名
  • メールアドレス・電話番号・住所
  • アカウント登録情報（ユーザー名・パスワードのハッシュ値）
  • お問い合わせ内容・フィードバック
  • 製品・SDK のライセンス登録に関する情報

（2）自動的に収集される情報

  • IPアドレス・ブラウザの種類・オペレーティングシステム
  • アクセス日時・参照URL・閲覧ページ
  • Cookie・ウェブビーコンなどのトラッキング技術により収集される情報
  • SDK の利用統計データ（匿名化・集計済みのもの）

（3）第三者から取得する情報

  • 当社と提携するパートナー企業からの情報
  • GitHub などのオープンソースプラットフォームを通じた開発者情報（公開情報のみ）

第3条　個人情報の利用目的

当社は、収集した個人情報を以下の目的のために利用します。

  • 本サービスの提供・運営・改善
  • 製品・SDK のサポート、技術的な問い合わせへの対応
  • ライセンス管理・認証サービスの提供
  • 新製品・アップデート・技術情報に関するご案内（お客様の同意がある場合）
  • マーケティング分析・統計調査（個人を特定しない形での集計）
  • 不正アクセス・不正利用の検知・防止
  • 法令に基づく義務の履行
  • 採用選考に関する連絡・情報管理

第4条　個人情報の第三者への提供

当社は、以下の場合を除き、利用者の個人情報を第三者に提供しません。

  • 利用者本人の事前の同意がある場合
  • 法令に基づく場合（裁判所・捜査機関等からの適法な要請を含む）
  • 人の生命・身体・財産の保護のために必要な場合
  • 当社の業務を委託する受託者（クラウドサービス事業者・分析ツール提供者等）に対し、必要な範囲で委託する場合。この場合、当社は受託者に対し、個人情報の適切な管理を義務付けます。
  • 当社の事業の全部または一部が合併・分割・事業譲渡等により承継される場合

第5条　Cookie およびトラッキング技術

当社のウェブサイトは Cookie を使用しています。Cookie とは、ウェブサイトがブラウザに保存する小さなテキストファイルです。当社は以下の目的で Cookie を使用します。

  • ログイン状態の維持・セッション管理
  • ユーザー設定（テーマ・言語・地域設定）の保存
  • ウェブサイトのアクセス解析・パフォーマンス測定
  • 不正アクセスの検知

お使いのブラウザの設定から Cookie を無効にすることができますが、その場合、本サービスの一部機能が正常に動作しない場合があります。

第6条　個人情報の保管と安全管理

当社は、個人情報の漏洩・滅失・毀損を防止するため、以下の安全管理措置を講じています。

  • SSL/TLS による通信の暗号化
  • 保管データの AES-256 暗号化
  • アクセス権限の最小化（最小権限の原則）
  • 定期的なセキュリティ監査・脆弱性診断
  • 従業員に対する個人情報保護教育の実施
  • インシデント発生時の対応手順の整備

ただし、インターネット上の通信は完全な安全性を保証するものではなく、当社は合理的な範囲での安全管理措置を講じますが、完全な安全性を保証するものではありません。

第7条　個人情報の保管期間

当社は、個人情報を以下の期間保管します。

  • アカウント情報：アカウント削除の申請から30日間
  • 問い合わせ・サポート履歴：対応完了後3年間
  • ライセンス登録情報：ライセンス有効期間終了後5年間
  • アクセスログ：収集から12ヶ月間
  • 法令上の保管義務がある情報：法令に定める期間

第8条　利用者の権利

利用者は、当社が保有する自身の個人情報について、以下の権利を有します。

  • 開示請求：当社が保有する個人情報の内容を確認する権利
  • 訂正・追加・削除請求：内容が事実と異なる場合の訂正等を求める権利
  • 利用停止・消去請求：個人情報の利用停止・削除を求める権利
  • 第三者提供停止請求：第三者への提供の停止を求める権利
  • データポータビリティ：当社が保有するデータを機械可読な形式で受け取る権利

これらの権利を行使する場合は、下記の連絡先までご連絡ください。本人確認手続きを経て、遅滞なく対応いたします（原則として受領から30日以内）。

第9条　未成年者の個人情報

本サービスは、16歳未満の方を対象とするものではありません。16歳未満の方が本サービスを利用する場合には、保護者の方の同意が必要です。16歳未満の方から個人情報を収集していることが判明した場合、当社は速やかに当該情報を削除します。

第10条　国際的なデータ転送

当社は、日本国内のデータセンターで個人情報を管理することを原則としますが、業務上の必要性から、日本国外に所在する当社の関連会社・業務委託先・クラウドサービス事業者にデータを転送する場合があります。その際は、適切なデータ保護措置（標準契約条項の締結等）を確保した上で行います。

第11条　ポリシーの変更

当社は、法令の改正・事業内容の変更・その他の事情により、本ポリシーを変更する場合があります。重要な変更を行う場合は、本ウェブサイト上での告知またはメールにより事前にお知らせします。変更後のポリシーは、当社ウェブサイトへの掲載をもって効力を生じます。継続してサービスをご利用いただいた場合、変更後のポリシーに同意したものとみなします。
