v2.0.0
2026年5月30日

大規模リニューアル — 日本語完全対応・新製品・スマホ最適化

新機能
新製品 X1 Air 追加 — 超薄型ノート向けの新チップ。8コア CPU（6P+2E）、12コア GPU、わずか 8W の TDP。省電力と薄型設計を両立し、学生・ビジネスパーソン向けエントリーラインとして新設。コンシューマーページに製品カード・スペックを追加。
新機能
リリースノートページ新設 — 本ページ（changelog/index.html）。ウェブサイト更新のたびに変更内容を具体的に記録します。バージョン番号・日付・タグ付き変更項目を掲載。
新機能
スマホ向けボトムナビゲーション実装 — 768px 以下の端末でフッター固定のボトムナビバーを表示。ホーム・製品・テクノロジー・スペックの各セクションにタップ1回でアクセス可能。Safe Area 対応済み。スクロール位置に応じてアクティブ状態が自動更新。
改善
完全日本語化 — コンシューマーページ（consumer/index.html）を含む全ページを完全日本語化。製品名・説明文・ナビゲーション・フッター・スペック表のすべてを日本語に統一。lang="ja"属性を全ページに設定済み。
改善
ライトモード完全実装 — システム設定（prefers-color-scheme）を自動検出し、初回アクセス時に適切なテーマを適用。手動切替後は localStorage に保存。製品カード・ベンチマーク・スペック表・ボタン・ナビゲーションすべてのライトモード対応を強化。テーマトグルボタンのアクセシブルな aria-label をテーマ変更時に動的更新。
改善
スマホ UI 全体的な最適化 — タッチターゲット最小 44px を全インタラクティブ要素に適用。iPhone のノッチ・Dynamic Island 対応（viewport-fit=cover、env(safe-area-inset-*)）。ボトムナビとコンテンツが重ならないよう body に padding-bottom を自動付与。
改善
価格セレクターのスタイル統合 — コンシューマーページの価格セレクター CSS を style.css に統合。`.price-selector`・`.price-tier-btn`・`.price-display` クラスを共通スタイルシートで管理するよう変更。
修正
HTML ファイルが空だった問題を修正 — 初回コミット時に index.html・consumer/index.html・enterprise/index.html が空文字列でプッシュされていた問題を修正。すべてのページに実際のコンテンツを書き込み再プッシュ（コミット 692b2b88）。
性能
システムダークモード変更の自動追従 — window.matchMedia の change イベントを監視し、ユーザーが OS のダーク/ライト設定を変更した場合、ページを再読み込みせずにリアルタイムでテーマを切り替え。ただし手動でテーマを変更した場合は localStorage の設定を優先。
v1.0.0
2026年5月30日

初回リリース — マルチページアーキテクチャ展開

新機能
ハブページ（index.html）新設 — スプリットパネル（コンシューマー/エンタープライズ）、ダイ系譜 Canvas アニメーション（X1 Pro → X1 Max → X1 Ultra）、ヒーロー、メトリクス、テクノロジー、ベンチマーク、スペックタブを含む日本語ハブページを構築。
新機能
コンシューマーページ（consumer/index.html）新設 — X1 Pro と X1 Max の製品ページ。スクロールストーリー（CSS Scroll-driven Animations + JS フォールバック）、価格セレクター、ベンチマーク比較を実装。
新機能
エンタープライズページ（enterprise/index.html）新設 — X1 Ultra と AX DataCenter の製品ページ。ROI 計算機（aria-live 対応、300ms デバウンス）、ケーススタディ3件、スペック比較表を実装。
新機能
イースターエッグ実装 — コナミコマンド（↑↑↓↓←→←→BA）でシークレットポップアップ表示。HTML ソースに隠しコメント3箇所。
新機能
iPhone 最適化 — viewport-fit=cover、apple-mobile-web-app-capable、Safe Area インセット対応、navigator.vibrate(1) ハプティックフィードバックを実装。
性能
will-change ライフサイクル管理 — CSS に静的な will-change を一切書かず、IntersectionObserver でビューポート進入時に付与・transitionend で除去。モバイル GPU レイヤー爆発を防止。
性能
prefers-reduced-motion 完全対応 — CSS @media (prefers-reduced-motion: reduce) ですべてのアニメーションを無効化。JS 側でも window.matchMedia チェックを実装しカウンターアニメーションをスキップ。
