はじめてのガイド

AXIOM SDK を使って最初のプログラムを動かすまでのステップ。

01

SDK をインストールする

パッケージマネージャーを使って AXIOM SDK をインストールします。macOS・Linux・Windows に対応しています。

# Homebrew (macOS / Linux)
brew install axiom-silicon/tap/axiom-sdk

# pip (Python)
pip install axiom-sdk

# npm (JavaScript / Node.js)
npm install @axiom-silicon/sdk
02

デバイスを列挙する

接続された AXIOM デバイスを検出し、利用可能なコアと Neural Engine のスペックを確認します。

import axiom_sdk as ax

# 利用可能なデバイスを列挙
devices = ax.list_devices()

for dev in devices:
    print(f"デバイス: {dev.name}")
    print(f"  CPU コア数  : {dev.cpu_cores}")
    print(f"  Neural TOPS: {dev.neural_tops}")
    print(f"  GPU コア数  : {dev.gpu_cores}")
03

Neural Engine でモデルを実行する

ONNX モデルをロードし、AX Neural SDK を使って Neural Engine 上で推論を実行します。

import ax_neural

# Neural Engine セッションを初期化
session = ax_neural.InferenceSession(
    model_path="model.onnx",
    device="neural",       # Neural Engine を指定
    precision="int8",       # INT8 量子化
)

# 入力テンソルを用意して推論実行
input_data = ax_neural.Tensor(shape=[1, 3, 224, 224])
output = session.run(input_data)

print(f"推論結果: {output.argmax()}")
print(f"レイテンシ: {session.last_latency_ms:.2f} ms")
04

GPU コンピュートカーネルを実行する

AX Graphics API を使って AX GPU G1 上でカスタムコンピュートシェーダーを実行します。

import ax_graphics as axg

# GPU デバイスを取得
gpu = axg.get_device("ax_gpu_g1")

# コンピュートパイプラインを作成
pipeline = gpu.create_compute_pipeline(
    shader="kernels/vector_add.axsl"
)

# バッファを割り当てる
buf_a = gpu.create_buffer(size=1024 * 4)
buf_b = gpu.create_buffer(size=1024 * 4)
buf_out = gpu.create_buffer(size=1024 * 4)

# ディスパッチ
cmd = gpu.command_buffer()
cmd.dispatch(pipeline, groups=(64, 1, 1),
             bindings=[buf_a, buf_b, buf_out])
gpu.submit(cmd)

# AX Neural SDK — モデル変換の例
import ax_neural.compiler as axnc

# PyTorch モデルを Neural Engine 形式へ変換
compiled = axnc.compile_torch(
    model=my_pytorch_model,
    input_shapes={
        "input": [1, 3, 224, 224]
    },
    precision="int8",
    target="ax_neural_v2",
    optimizations=[
        "fuse_bn",
        "fold_constants",
        "prune_unused",
    ]
)

# コンパイル済みモデルを保存
compiled.save("model_ax.axnn")
print(f"最適化後サイズ: {compiled.size_mb:.1f} MB")
print(f"推定 TOPS 使用率: {compiled.tops_util:.1%}")
