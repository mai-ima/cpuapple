AX GPU G1 / G1 Pro 発表 — 2nm 採用独立 GPU で AI レンダリングの概念を塗り替える

AXIOM Silicon は2026年5月、同社初の独立型 GPU「AX GPU G1」および「AX GPU G1 Pro」を正式発表しました。TSMC N2 プロセスを採用した両製品は、Neural Engine との緊密な統合により、レンダリングと AI 推論を同一ダイ上で並列実行。G1 Pro は 76 TOPS の Neural Engine と 8,192 シェーダーコアを搭載し、プロフェッショナルクリエイターとゲーマー双方の要求に応えます。価格・発売日は今後発表予定。

2nm プロセスノードの量産を開始 — TSMC N2 採用、業界最高トランジスタ密度を実現

AXIOM Silicon は TSMC との連携のもと、N2 プロセス（2nm 相当）での量産を2025年10月より正式に開始したと発表しました。1億6500万個/mm² のトランジスタ密度を達成し、X1 Ultra を皮切りに全製品ラインへ順次展開します。

シリーズ B ラウンドで300億円を調達 — 次世代 GPU および Mobile SoC の開発を加速

AXIOM Silicon は2025年6月、シリーズ B ラウンドで総額300億円の資金調達を完了したと発表しました。本ラウンドは国内外の主要 VC および戦略的投資家が参加。調達資金は AX GPU G1 シリーズの開発加速および AX Mobile M1（スマートフォン SoC）の設計・量産準備に充当されます。
