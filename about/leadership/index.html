AXIOM Silicon の経営チームは、半導体・ソフトウェア・金融の各分野における深い専門知識と、変革への情熱を持つリーダーたちで構成されています。

山本 海斗
共同創業者・チーフアーキテクト

東京大学情報理工学系研究科修了。前職はTSMCの研究開発部門でプロセス設計を担当。2021年にAXIOM Siliconを共同創業し、X1 アーキテクチャおよび UltraFabric インターコネクト技術を設計。半導体業界における日本発イノベーションの象徴として国内外から注目を集めている。

鈴木 奈々
CTO

スタンフォード大学コンピュータサイエンス博士。NVIDIA のアーキテクチャチームで GPU 設計に10年間従事した後、AXIOM Silicon に参画。AX GPU G1 シリーズおよび Neural Engine の技術責任者として、世界最高水準の演算効率を実現するシリコン設計を主導している。

田中 拓海
CFO

慶應義塾大学経済学部卒業後、ゴールドマン・サックス証券にてテクノロジー・セクターの投資銀行業務に従事。シリーズ A・B の資金調達を主導し、累計300億円超の資本調達を実現。AXIOM Silicon の財務戦略・IR・M&A 活動を統括し、持続的な成長基盤を構築している。

渡辺 美桜
VP Engineering

京都大学工学研究科電気工学専攻修了。ARM Holdings にてモバイル SoC の設計チームを率いた後、AXIOM Silicon に入社。AX Mobile M1 および AX Memory シリーズの開発を統括。400名超のエンジニアリング組織を牽引し、製品の設計から量産まで一貫した品質管理体制を確立している。
